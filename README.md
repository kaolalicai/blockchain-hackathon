### 我们想解决什么问题

解决点：

 - 金融征信孤岛：企业数据共享上链
 - 数据获取成本高、渠道分散并有限：公开 blockchain 直接读取
 - 数据隐私保护：对ID类信息（如身份证等）单向hash后上链(类同态加密)
 - 数据质量：真实有用数据加token，假数据扣减token（token不足不再允许读写）
